# DIU CPC Website

A beautiful, dynamic website for the DIU Competitive Programming Club (CPC) featuring glassmorphism design and interactive elements.

## Features

- **Glassmorphism UI**: Modern glass-like effects with skyblue and white theme
- **Dynamic Content**: Events loaded from JSON, interactive gallery with modal
- **Responsive Design**: Works on all devices
- **Theme Toggle**: Switch between light and dark modes
- **Form Validation**: Contact form with real-time validation
- **Smooth Animations**: Scroll effects and transitions

## Pages

- **Home**: Hero section with news ticker and club overview
- **About**: Club history, mission, timeline, and advisors
- **Events**: Dynamic event listings with registration links
- **Gallery**: Photo carousel with modal lightbox
- **Team**: Committee members with social links
- **Resources**: Tutorials and programming tools
- **Contact**: Contact form and location map

## Technologies Used

- HTML5
- CSS3 (with Bootstrap)
- JavaScript (ES6+)
- Font Awesome Icons
- JSON for data storage

## Local Development

1. Clone the repository
2. Open `index.html` in your browser, or
3. Run a local server:
   ```bash
   python -m http.server 8000
   ```
   Then visit `http://localhost:8000`

## Deployment

This site is ready for deployment on GitHub Pages:

1. Create a new repository on GitHub
2. Push this code to the repository
3. Go to Settings > Pages
4. Select "Deploy from a branch" and choose `main` branch
5. The site will be live at `https://yourusername.github.io/repository-name/`
