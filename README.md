# DIU CPC Website

This is a static website for the Dhaka International University Computer Programming Club (DIU CPC).

## Pages
- Home (index.html)
- About (about.html)
- Events (events.html)
- Gallery (gallery.html)
- Team (team.html)
- Resources (resources.html)
- Contact (contact.html)

## Technologies
- HTML
- CSS (Bootstrap 5)
- JavaScript

## Deployment
Host on GitHub Pages:
1. Create a GitHub repository.
2. Push these files.
3. Enable Pages in repository settings.
